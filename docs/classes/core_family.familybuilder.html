Class FamilyBuilder

A builder pattern to create Family objects. Use Family.all(), Family.one() or Family.exclude() to start

Example usage:

let family = Family.all(ComponentA, ComponentB).one(ComponentC, ComponentD).exclude(ComponentE).get();
