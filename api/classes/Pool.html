# Methods

# free

▸ free(object): void

Puts the specified object in the pool, making it eligible to be returned by {@link obtain}. If the pool already contains {@link max} free objects, the specified object is not added to the pool. The pool does not check if an object is already freed, so the same object must not be freed multiple times.
